TurboWarp Packager Legacy Versions

These are old versions of the TurboWarp Packager. While we think that every new version of the packager is better than the previous, we understand there are cases where you might want to use an old version.

We will keep these versions up and in (mostly) working condition for as long as possible, but we can't provide any support for non-critical bugs or feature requests.

apache (February 23, 2024)

This is a snapshot of the packager before we changed its license from the Apache 2.0 license to the GNU Lesser General Public License version 3 (LGPLv3).

While don't expect the LGPL to cause any problems, even for people who are selling packaged projects, there might be some legal edge cases where it is a problem. We're keeping this around for them.

p3 (June 23, 2021)

This is the old version of packager before we rewrote it all from the ground-up. The name "p3" comes from the internal name we gave it: "packager v3". The first two versions were the forkphorus packager, which itself was rewritten once.

p3 is notable for using TurboWarp/scratch-gui as the basis for packaged projects, which subjected packaged projects to the full GNU General Public License v3, and made the projects significantly slower, bigger, and less customizable than the rewrite.
